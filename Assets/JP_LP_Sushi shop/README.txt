Low Poly Japanese Sushi Shop <Created by sbbututuya>
--------------------


Description:
--------------------
With this package you can make a "Japanese sushi shop".

This package contains:

- 347 meshs. (Polygon count : Min 2 - Max 2017)
- 438 prefabs.
- 7 materials.
- 4 textures. (1024x1024/2048x2048)
- 1 skybox procedural material.
- 1 demo scene.
- 1 post processing stack v2 profile.

Demo scene includes helper scripts for conveyor belt sushi.

Been used for AssetStore screenshots visualization:

- Post Processing Stack V2
